# i-vagrant
i-vagrant helps set up a development environment easily.

Try to fork this repository and build i-vagrant for own needs.

## Getting Started Guide
Download i-vagrant
```bash
$ git clone https://github.com/bogdanvlviv/i-vagrant.git
$ cd i-vagrant/
```
